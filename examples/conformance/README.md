# Conformance Fixtures

This tree holds non-canonical conformance fixtures used by validator and CLI tests.

- The canonical valid examples remain under `examples/document-basic/`, `examples/comic-basic/`, and `examples/storyboard-basic/`.
- The fixtures here are intentionally invalid package directories used to lock stable failure behavior.
- They are not part of `pnpm examples:validate` or `pnpm examples:smoke`.
